# Ovin: Intelligent Sheep Farming Platform

## Description
Ovin is an intelligent sheep farming platform designed to enhance farm management through technology. It provides real-time data analytics, monitoring, and insights to optimize sheep farming operations.

## Features
- Real-time health monitoring of sheep.
- Predictive analytics for sheep growth.
- Automated feeding schedules.
- User-friendly dashboard for farm management.

## Architecture
The architecture of Ovin consists of several components:
- Sensor Network for health monitoring.
- Cloud Database for data storage.
- Web Application for user interaction.

## Setup Instructions

### Prerequisites
- Node.js
- MongoDB
- Docker (Optional)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/nacerouali464-sketch/Ovin.git
   cd Ovin
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Configuration
- Create a `.env` file and configure the necessary environment variables.

## Deployment Guide
1. Build the application:
   ```bash
   npm run build
   ```
2. Deploy using Docker:
   ```bash
   docker-compose up -d
   ```

## Usage
- Access the web app at `http://localhost:3000`.
- Follow the on-screen instructions to get started.
